# Components

This directory contains all React components organized by feature area.

## Structure

- `ui/` - Shadcn/UI base components (Button, Input, Card, etc.)
- `auth/` - Authentication-related components (LoginForm, SignupForm, etc.)
- `student/` - Student-specific components (ChatInterface, MessageBubble, etc.)
- `professor/` - Professor-specific components (Dashboard, AnnouncementDrafts, etc.)
- `shared/` - Shared components used across the app (FileUpload, LoadingSpinner, ErrorBoundary, etc.)
