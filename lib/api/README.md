# API Client Functions

This directory contains API client functions for communicating with the backend.

## Structure

- `chat.ts` - Chat API calls
- `upload.ts` - Upload API calls
- `escalations.ts` - Escalations API calls
- `announcements.ts` - Announcements API calls
- `conductor.ts` - Conductor API calls
- `pulse.ts` - Pulse API calls

## Usage

All API clients should use the base API client utility (to be created) for consistent error handling and authentication.
